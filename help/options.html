Настройки приложения

Диалог настройки

Вызов диалога для настройки приложения производится щелчком на пункте меню [Инструменты]->[Опции]. После этого на экране появляется окно, изображенное на рисунке 1.

Приложение имеет минимальное количество настроек. Вы можете по своему усмотрению выбрать каталог (папку) для хранения файлов с данными. По умолчанию установлен каталог "data", который находится в каталоге с установленным приложением. Когда вы начинаете новый учет, созданный файл с данными помещается в эту папку.

Можно выбрать язык интерфейса приложения - русский или английский.

По умолчанию скрыты кнопки редактирования таблицы "Суммарные данные за день". Так как дни учета добавляются в таблицу последовательно, то к редактированию таблицы вручную (изменению даты учета, удалению дня учета) следует прибегать в крайних случаях. Тем не мение можно показать скрытые кнопки в текущем сеансе использования приложения, поставив флажок в соответствующем поле.

На рисунке 2 показаны фрагменты главного окна приложения со скрытыми a) и отображаемыми b) кнопками редактирования таблицы "Суммарные данные за день".
