Начало учета

Первый запуск приложения

После первого запуска приложения на экране появляется главное окно с открытым демонстрационным файлом (см. рисунок 1).

Для начала использования приложения нужно создать новый файл в котором будут храниться введенные вами данные. Для этого следует выбрать пункт меню (меню расположено вверху главного окна) [Файл]->[Новый].

На экране появится форма (см. рисунок 2) для ввода персональных данных, в которую нужно ввести имя, пол, дату рождения, рост, дату начала учета, текущий вес, активность и щелкнуть на кнопке «Далее» (кнопка «Далее» будет недоступной до тех пор, пока вы не заполните поле "Имя").

Затем следует указать имя файла и путь в поле, обозначенном на рисунке 3 цифрой [1]. Для этого удобно воспользоваться стандартным диалогом операционной системы, щелкнув на кнопке [2].

В появившемся диалоговом окне (см. рисунок 4) следует ввести имя файла, например mydata, и щелкнуть на кнопке «Сохранить».

После этого нужно щелкнуть на кнопке «Завершить» (см. рисунок 3). На экране появится окошко, показывающее ход создания меню продуктов и блюд (см. рисунок 5). Дождитесь завершения создания меню (окошко автоматически закроется).

После завершения создания меню приложение готово к вводу ваших данных (см. рисунок 6).

В таблице "Суммарные данные за день" присутствует одна строка (запись) с датой начала учета и вашим весом. Таблица "Пища съеденная ..." пока пуста. Пусты и графики (диаграммы). Они будут отображать линии после ввода данных хотя бы за два дня.

Пример ввода данных

Допустим в первый день учета ваш завтрак состоялся в 7:00 и включал следующие продукты: овсянка на молоке - 100 г, хлеб бородинский - 35 г, кофе растворимый без сахара - 150 г. Щелкаем на кнопке [6](см. рисунок 6), на экране появляется окно меню для выбора продуктов или блюд (см. рисунок 7). Это окно можно сдвинуть так, чтобы оно не перекрывало главное окно приложения и растянуть для удобных размеров.

В меню продуктов [1] найдите и выделите запись «овсянка на молоке». В поле «Время употребления» [4] введите 7:00. В поле «Количество, г» [5] введите 100 и щелкните на кнопке «Добавить». В таблицу "Пища съеденная..." добавится запись.

Найдите и выделите в меню «хлеб бородинский», введите число 35 в поле «Количество, г» и снова щелкните «Добавить» - добавится еще одна запись.

Найдите и выделите «кофе растворимый», занесите данные о нем и опять щелкните «Добавить».

Данные о завтраке занесены. Чтобы закрыть окно выбора продуктов, щелкните на кнопке «Закрыть». После добавления данных о завтраке, главное окно приложения примет вид, показанный на рисунке 8.

Аналогично заносятся данные о последующих приемах пищи.
