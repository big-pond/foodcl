Ввод и редактирование данных

Ввод дня (даты) учета и веса

Даты учета вводятся последовательно, для этого предназначена кнопка [5]. При щелчке на этой кнопке происходит добавление строки в таблицу [7] "Суммарные данные за день". При этом в первую ячейку добавленной строки автоматически записывается следующая дата. Например если добавить строку в таблицу изображенную на рисунке 1, то в первой ячейке строки будет дата "2020-04-12". Даты в приложении отображаются в виде "год-месяц-день".

Следует заметить, что значения в ячейках таблицы "Вес съеденного, г" и "Калорийность съеденного, ккал" рассчитываются в приложении автоматически и не редактируются вручную. Как показал опыт использования приложения, кнопки [1], [2], [3], [4], предназначенные для навигации по строкам таблицы, используются редко, так как для этого удобнее пользоваться полосой вертикальной прокрутки таблицы.

Для ввода собственного веса следует выделить строку с нужной датой и щелкнуть на кнопке [6], на экране появится диалоговое окно, изображенное на рисунке 2.

Далее нужно ввести значение веса в соответствующее поле и щелкнуть на кнопке "ОК". В ячейке "Мой вес, кг", выделенной строки, появиться введенное значение.

Для редактирования ошибочно введенного значения веса следует выполнить такие же действия.

Вводить значение собственного веса на каждую дату учета не обязательно. Если вы не взвешивались - оставьте пустой ячейку строки "Мой вес".

Ввод и редактирование данных о съеденной пище

После добавления следующей даты в таблицу "Суммарные данные за день" строка с этой датой становится текущей и можно вводить данные о пище, съеденной в этот день. Нужно отметить, что после запуска приложения последняя строка этой таблицы всегда становится текущей.

В приложении не производится разделение приемов пищи на завтрак, обед и ужин, а просто вводится примерное время употребление пищи. Например, если завтрак был 7:00, то можно указать это время для всех съеденных на завтрак продуктов.

Добавление записей о приеме пищи

Для добавления новой записи о приеме пищи нужно щелкнуть на кнопке [5], изображенной на рисунке 3.

На экране появится диалоговое окно (см. рисунок 4) для выбора продукта (блюда), указания времени употребления и его веса. После выбора продукта из меню, ввода значений времени и веса следует щелкнуть на кнопке "Добавить" и новая строка с введенными значениями добавится в таблицу [8] (см. рисунок 3). При этом диалоговое окно не закрывается, а ожидает выбора следующего продукта и ввода его времени употребления и веса.

Для поиска продукта (блюда) в меню можно воспользоваться полем "Поиск", в котором нужно набрать название продукта. В ходе набора текста, в меню будет выделяться продукт (блюдо), название которого наиболее полно совпадает с введенной последовательностью букв.

Для быстрого перемещения между полями ввода можно испльзовать "горячие" клавиши. Сочетание Ctrl+F переносить фокус ввода в поле "Поиск", Ctrl+T - в поле "Время употребления" и Ctrl+W - в поле "Вес, г".

В верхней части окна расположены кнопки, которые позволяют редактировать меню не прекращая ввода данных о приемах пищи. С описанием процесса редактирования меню можно познакомиться здесь

После ввода данных о приемах пищи следует закрыть окно щелкнув на кнопке "Закрыть".

Следует заметить, что редактирование приемов пищи производится в так называемом модальном режиме, т.е. в это время блокируется доступ к главному окну приложения и его атрибутам (меню, кнопкам и т.п.).

Редактирование записи о приеме пищи

Для редактирования записи о приеме пищи (см. рисунок 5) нужно выбрать ее в таблице [8] и щелкнуть на кнопке [6].

На экране появится диалоговое окно (см. рисунок 4) такое же, как и для добавления приемов пищи за исключением кнопок в его нижней части, которые сменили надписи на "ОК" и "Отмена". Отредактировав значения соответствующих полей следует щелкнуть на кнопке "ОК" для внесения изменений в выделенную строку таблицы [8] или "Отмена" если решили не вносить изменения. После этого диалоговое окно закроется.

Удаление записи о приеме пищи

Для удаления записи о приеме пищи (см. рисунок 6) нужно выбрать ее в таблице [8] и щелкнуть на кнопке [7].

Будьте внимательны - удаление записи происходит без запроса на подтверждение удаления.

Примечание

Диалоговое окно для добавления и редактирования приемов пищи "запоминает" свое положение на экране. Поэтому его можно расположить справа от главного окна приложения так, чтобы оно не перекрывало его. Такое положение позволяет видеть изменения в таблицах и на графиках непосредственно при вводе новых данных или их редактировании (см. рисунок 7).
