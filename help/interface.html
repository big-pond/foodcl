Интерфейс приложения

Главное меню приложения

На рисунке 1 изображено главное меню приложения foodcl.

Меню "Файл"

Пункт меню "Открыть" используется для открытия файла учета. После щелчка на пункте на экране появляется стандартный диалог операционной системы для выбора и открытия файла. После завершения сеанса работы с приложением и выхода из него, путь к файлу будет запомнен и файл автоматически откроется при следующем запуске приложения.

Пункт меню "Новый" используется для создания нового файла учета. Щелчок на пункте приводит к вызову последовательности диалоговых окон, в которые нужно ввести персональные данные, имя файла и, если необходимо, заполнить меню из старого файла учета, указав путь к этому файлу. Подробно создание нового файла учета описано здесь.

Список файлов служит для быстрого открытия файлов, в которые ранее вводились данные учета. В списке хранится до пяти файлов. Первым в списке располагается файл, с которым пользователь работает в данный момент времени.

Пункт меню "Выход" закрывает приложение, при этом нет необходимости заботится о сохранении данных, введенных в течение сеанса. Приложение выполняет сохранение информации автоматически.

Меню "Вид"

Пункт меню "Графики" скрывает или отображает область с графиками "Калории" и "Вес", которая расположена в правой части главного окна приложения.

Пункт меню "Строка статуса" скрывает или отображает строку в нижней части главного окна, которая содержит информацию об авторских правах на приложение.

Меню "Информация"

Пункт меню "Меню продуктов (блюд)" выводит на экран окно для редактирования меню продуктов и блюд.

Пункт меню "Персональные показатели" выводит на экран диалоговое окно для просмотра и редактирования персональных показателей.

Меню "Запросы"

Пункт меню "Количество съеденной пищи" выводит на экран окно для просмотра общего количества употребленной пищи за заданный период времени. Описание размещено на этой страничке.

Пункт меню "Даты употребления продукта" выводит на экран окно для просмотра употребления выбранного продукта или блюда по датам. Подробнее здесь.

Пункт меню "Соотношение белков, жиров и углеводов" выводит на экран окно для просмотра соотношения белков жиров и углеводов в вашем питании. Подробнее на этой страничке.

Меню "Инструменты"

Пункт меню "Индекс массы тела" выводит на экран окно для расчета индекса массы тела, основного обмена (базальной скорости метаболизма), общего расхода энергии за сутки и идеального веса. Окно появляется с персональными данными, находящимися в открытом в данный момент файле. Но перечисленные показатели можно рассчитать и для любого человека, вводя его данные. Подробнее здесь.

Пункт меню "Опции" выводит на экран диалоговое окно для изменения настроек приложения. Описание находится на этой страничке.

Меню "?"

Пункт меню "Помощь" открывает в браузере страничку "Введение" описания приложения foodcl.

Пункт меню "О программе" выводит на экран окно с краткой информацией о приложении foodcl.

Пункт меню "О Qt" выводит на экран окно с информацией о библиотеке Qt, которая использовалась для разработки приложения.
