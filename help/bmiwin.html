Расчет персональных показателей

Для расчета персональных показателей следует щелкнуть на пункте меню [Инструменты]->[Индекс массы тела]. После этого на экране появляется окно, изображенное на рисунке 1, в котором можно рассчитать:

  • индекс массы тела;
  • основной обмен (базальную скорость метаболизма) по формуле Миффлина-Джеора
  • общее расходование энергии за сутки
  • диапазон идеального веса

Окно появляется с персональными показателями пользователя, которые хранятся в открытом в данный момент файле. Однако перечисленные показатели можно рассчитать для любого человека, используя его данные. Для этого надо отредактировать значения полей: "Возраст", "Пол", "Рост", "Вес" и "Активность". В таблице "Результаты расчета" появятся новые значения показателей.

Диапазон идеального веса рассчитывается с учетом интервала индекса массы тела, который в норме составляет 18,5 - 25 кг/м².
